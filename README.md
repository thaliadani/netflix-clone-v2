## 📌 Sobre o Projeto

Este projeto é uma recriação da interface da Netflix, focando na semântica do **HTML5** e no poder de estilização do **SCSS**. O objetivo principal foi aplicar conceitos avançados de **Flexbox**, **CSS Grid** e **Responsividade** para garantir que a interface seja fluida em qualquer dispositivo.

## 🚀 Funcionalidades

- [x] Layout totalmente responsivo (Mobile, Tablet e Desktop).
- [x] Efeitos de hover interativos nos cards de títulos.
- [x] Navegação estilizada com Header fixo.
- [x] Organização de estilos utilizando a arquitetura modular do SCSS.

## 🛠️ Tecnologias

- **HTML5**: Estruturação semântica do conteúdo.
- **SCSS (Sass)**: Utilização de variáveis, mixins e aninhamento para um CSS mais limpo e escalável.
- **VS Code**: Ambiente de desenvolvimento.

## 🏁 Como Executar

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/thaliadani/netflix-clone-v2.git
   ```
2. **Acesse a pasta do projeto:**
   ```bash
   cd netflix-clone-v2
   ```
3. **Abra o projeto:**
   Basta abrir o arquivo `index.html` em seu navegador ou utilizar a extensão **Live Server** no VS Code.
